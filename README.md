## Description
This Java program models Gold, Copper, and Coffee futures contracts.
The program reads historical price data from text files, stores the prices in an ArrayList, and calculates volatility using a Monte Carlo simulation.
The assignment demonstrates the use of interfaces, inheritance, exception handling, file reading, ArrayLists, and random sampling.
Historical price data for Gold, Copper, and Coffee was obtained from Investing.com.

## Java Files
- FutureContract.java
- IVolatilityCalculator.java
- MonteCarloVolatility.java
- GoldFuture.java
- CopperFuture.java
- CoffeeFuture.java
- FuturesTest.java

## Data Files
- gold_prices.txt
- copper_prices.txt
- coffee_prices.txt
The data files contain historical daily prices for Gold Futures, Copper Futures, and US Coffee C Futures.
The price data covers July 9, 2026 through July 22, 2026.

## How to Run
1. Open the project in OnlineGDB.
2. Make sure all Java files and text data files are included in the project.
3. Run the program using FuturesTest.java.
4. The program will load the historical prices from the text files.
5. The program will calculate and display the volatility for Gold, Copper, and Coffee futures.
